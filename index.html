Welcome to the XModeler distribution site.

On this page the monthly builds of the Xmodeler are available.

Simply follow the download links given above.

If you're interest in pushing the development of Xmodeler, simply clone the repository with 


$ git clone git://github.com/phalix/XModeler.git

The source code is further backed up at the "CurrentBackup"-Tag in


$ git clone ssh://git@bitbucket.org/xmodeler/xmodeler.git

After the checkout run 


$ cd xmodelermaven
$ mvn install

within the the root directory.

Dependencies will automatically resolved based on the indigo release of eclipse.

Monthly builds are realised through CI with a Jenkins Server hosted on CloudeBees. Builds are currently only created monthly, due to resource restrictions.
